Case Overview

▼
What is “Data Leakage”?

Think of it like this: Imagine your school has a secret recipe book locked in a cabinet. Data leakage is when someone sneaks in, copies all the recipes, and gives them to a competitor school — without permission. In this case, “Iaman Informant” was an employee who secretly copied confidential project files and sent them to an outside contact. They then tried to hide their tracks by deleting files and wiping their computer.

What is Eraser?

Eraser is a program that deletes files in a special way. Normal deletion just marks space as “available” — the data is still there and can be recovered. Eraser overwrites the data with random ones and zeros multiple times, making it impossible to recover. Using this on the last day of system activity strongly suggests deliberate destruction of evidence.

What is CCleaner?

CCleaner is a “system cleaning” tool that removes browser history, temporary files, and clears Windows registry traces. The suspect installed it, ran it, then UNINSTALLED it — trying to hide the fact it was ever used.

  • Installed Google Drive on 2015-03-23
  • Used PERSONAL Gmail account (iaman.informant.personal@gmail.com) — not the work account — making it harder to trace

Glossary of Terms

▼

Not sure what something means? Here is a simple explanation of all technical terms used in this report.

Hash / MD5 / SHA-1
A unique “fingerprint” number for a file. If even one byte changes, the fingerprint changes completely. Used to verify the disk image was not tampered with.
Disk Image (DD)
An exact copy of an entire hard drive — like a perfect photograph of all the data.
NTFS
The file system Windows uses to organize files on a hard drive (like a filing system).
Partition
A section of a hard drive — like dividing a big room into separate areas.
Registry Hive
A special database Windows uses to store all its settings (like a control panel for the whole computer).
DHCP
A system that automatically assigns an IP address (a network identity number) to a computer.
IP Address
A unique number that identifies a computer on a network — like a home address for a computer.
Forensic Artifact
A digital trace left behind by computer activity — like footprints in the snow.
USN Journal ($UsnJrnl)
A log kept by Windows that records every file create, delete, rename, or modify operation.
Prefetch
A Windows feature that records when programs are run, to help them start faster. Also useful for forensics.
UserAssist
A hidden Windows record of which programs the user has run, and how many times.
ShellBag
A registry record of which folders the user has browsed in Windows Explorer.
JumpList
A Windows feature that records recently opened files per application.
LNK file
A Windows shortcut file. Created automatically when a file is opened — reveals file access times.
OST file
Outlook’s local copy of an email mailbox. Can be analyzed to recover emails.
ESE Database
A type of database format used by Windows and IE11 to store browser history and other data.
Thumbcache
A cache of small preview images (thumbnails) of files viewed in Windows Explorer. Survives file deletion.
Anti-forensics
Techniques used to hide, destroy, or confuse digital evidence — like erasing footprints.
Eraser
A program that permanently destroys files by overwriting them, making recovery impossible.
CCleaner
A “cleaning” program that removes browser history, temp files, and Windows registry entries.
IMAPI
Windows’ built-in CD/DVD burning system.
Google Drive Sync
A program that automatically uploads files in a folder to Google’s cloud storage.
WAL (Write-Ahead Log)
A backup file used by SQLite databases. Even when the main database was deleted, the WAL file retained the data.
MFT ($MFT)
Master File Table — the index of every file on an NTFS partition. Like a library catalog.
